# Magnolia CMS

A Magnolia CMS project with the **starter-lm** light module (Freemarker templates, components, and Java services).

## Prerequisites

- Node.js
- Maven + Java 21

## How to launch

1. **First-time setup** (downloads Magnolia runtime):
   ```bash
   npm run mgnl jumpstart
   ```

2. Mvn install
   ```bash
   mvn clean install
   ```

3. **Start Magnolia**:
   ```bash
   npm run mgnl start
   ```

4. Open **http://localhost:8080/magnoliaAuthor** in browser.
